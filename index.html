Quick Start

Last updated: Jan 25th, 2020

Installation

Step 1 Unboxing product and accessories
Step 2 Drive the nail into the wall.
Step 3 Hang on the wall with backside hook
Step 4 Plug in the USB cable

Download App

ClockClock App is a remote software to control the device, which could be downloaded by the following ways.

iOS version is coming soon.

Windows Mac Android iOS (coming soon)

Connect Wifi

Once ClockClock is powered on, search for the wifi point with name "ClockClock" and password "clockclock".

# Name Password
1 clockclock clockclock

App Interface

1. Task Panel

In the task panel, several modes are provided for users.

# Mode Description
1RebootRestart the device (this may reset some settings)
2PropellerTurn to ropeller style
3ClapTurn to clap style
4RightAngleTurn to rightangle style
5RotateRotate three times at current position
6OnlyTimeOnly show the time
# Button Action
1SendSend the selection mode to device                     
2PlayPausePlay or pause the device

2. Time Panel

In the time panel, user could set device time.

# Mode Description
1HourManually select the hour                      
2MinuteManually select the minute
# Button Action
1SendSend the selected time to device         
2SyncAuto sync and send network time

3. Tune Panel

In the tune panel, user could manually calibrate the clock hand by following steps.

# Step
1Turn on tune switch and wait 30 seconds to enter calibrate mode
2Tap and choose the clock need to be calibrated
3Click +/- button to adjust up/down hand position
4Turn off tune switch and device will exit calibrate mode
# Button Action
1Up +/-Adjust up hand by step
2Down +/-Adjust down hand by step
